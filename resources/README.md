# Resources Directory

This directory contains all static assets and sample data for the Mentor-Mentee Matchmaker project.

## Structure

- `assets/` - Static assets (CSS, images, etc.)
- `sample_data/` - Sample CSV files and test data
- `staticfiles/` - Collected static files (generated by Django)

## Asset Organization

Static assets are organized as follows:
- CSS files in `assets/`
- Images would go in `assets/images/` (if we had any)
- JavaScript files would go in `assets/js/` (if we had any)

## Sample Data

Sample data includes:
- Template CSV files for mentors and mentees
- Sample data files for testing
- Test data sets for different phases
